# Virtual Reality Driving Simulation for Driving Test Preparation

**Game Title:** Car-Noobies
**Type:** Non-Immersive Virtual Reality Driving Simulation
**Purpose:** Final Year Project (CSP650), Semester 6
**Institution:** Universiti Teknologi MARA (UiTM) Cawangan Melaka Kampus Jasin
**Supervisor:** Dr. Siti Feirusz Ahmad Fesol
**Built With:** Unity | C#

---

## Project Overview

Car-Noobies is a non-immersive virtual reality driving simulation application
designed to assist learners in preparing for the JPJ (Jabatan Pengangkutan Jalan)
driving test. The simulation replicates the official JPJ circuit test environment,
allowing learners to practise driving scenarios in a risk-free virtual setting
before their actual test.

---

## Features

- Third-person and first-person (cockpit) driving views
- Real-time rear view mirror and side mirror displays
- Live test progress tracker showing pass/fail status per component
- Countdown timer per test component
- Real-time driving guidance prompts in Bahasa Malaysia
- Traffic light detection and response scenarios
- Full JPJ Ujian Litar simulation with 5 test components

---

## JPJ Ujian Litar Components

| No | Component |
|----|-----------|
| 1 | Mendaki Bukit (Hill Climb) |
| 2 | Tiga Penjuru (Three-Point Turn) |
| 3 | Memarkir Sisi (Side Parking) |
| 4 | Selekoh S (S-Curve) |
| 5 | Selekoh Z (Z-Curve) |
